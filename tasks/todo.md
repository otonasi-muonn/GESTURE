# Gesture Show! ブラッシュアップ進行表

## 価値軸

- 対象: ジェスチャーゲームの出題・回答表示を行う送信機／受信機ユーザー
- 価値: 「決定」と「戻る」を迷わず使え、1 画面内で状態が伝わり、P2P 切断や役割交代から自力復旧できること
- 制約: 固定同期 ID、既存の MediaPipe Hands + PeerJS 構成、人差し指先端カーソルと待ち時間のない指構成入力を維持する。新規外部依存・ルームコードは追加しない。マウス／タッチ fallback は維持する。
