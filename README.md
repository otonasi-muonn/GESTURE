# Gesture Show! 🐼🖐️ (ジェスチャーゲーム回答表示アプリ)

カメラに向かって手を動かし、「グー」や「合掌」をするだけで操作できる、**非接触（タッチレス）型ジェスチャーゲーム回答表示ウェブボード**です。

出題者（演者）は画面から離れた場所から、指を使って画面上の正解判定や画面遷移を完全にタッチレスでコントロールできます。

---

## 🎮 操作方法 (ジェスチャーコントロール)

1. **ポインターを動かす (手を伸ばす)**
   * カメラに片手を映すと、画面上に光るサークル状の「手のひらカーソル」が出現します。
   * 手の動きに合わせてカーソルが滑らかに追従します。

2. **決定・選択 (グー ✊ にする)**
   * ボタンや1〜10のお題カードの上にカーソルを重ねて、指を握り込みます。
   * 約1秒間「グー」の状態をキープすると、ゲージがチャージされ選択が実行されます。
   * 正解したお題は、**背景色が明るいグリーンに変化**し、正解音が鳴ります。

3. **戻る (手と手を合わせる 🙏 - クラップ・合掌)**
   * お題一覧（詳細）画面からカテゴリー選択画面に戻りたいときは、カメラの前で**両手のひらをパッと合わせる（または近づける）**だけです。
   * 瞬時に「シュン！」と音が鳴り、ホーム画面に戻ります。

*※カメラが使えない場合やうまく反応しない場合は、マウスやタッチ操作でもすべての機能をご利用いただけます。*

## 🛠️ 技術スタック & 動作原理

* **Core**: HTML5, Vanilla CSS, Vanilla Javascript (ES6+)
* **手形トラッキング**: 
  * [MediaPipe Hands](https://developers.google.com/mediapipe/solutions/vision/hand_landmarker) (jsDelivr CDNから動的ロード)
  * Webカメラの映像ストリームからリアルタイムに両手最大21箇所の骨格座標（Landmarks）を取得。
  * `requestAnimationFrame` と線形補間 (Lerp) による、ブレのない滑らかなカーソル追従。
* **オーディオ**:
  * [Web Audio API](https://developer.mozilla.org/ja/docs/Web/API/Web_Audio_API) による内蔵シンセサイザー。
  * 外部mp3ファイルを使わず、ブラウザのオシレーターからリアルタイムで効果音を波形合成するため、ロードエラーが一切ありません。

## 🚀 GitHub Pages へのデプロイ方法

本リポジトリには GitHub Actions 用のワークフローが設定されているため、GitHub にプッシュするだけで自動的に公開されます。

### 手順：
1. 本プロジェクトのコードを GitHub リポジトリ（例: `your-username/gesture-show`）にプッシュします。
2. GitHub リポジトリの **Settings** ページを開きます。
3. 左メニューの **Pages** を選択します。
4. **Build and deployment** > **Source** で `GitHub Actions` を選択します。
5. プッシュ後、**Actions** タブでデプロイの進行状況が確認でき、完了すると専用URL（`https://your-username.github.io/gesture-show/`）で公開されます。

### ローカルでの動作確認：
MediaPipeはカメラ（`getUserMedia`）を使用するため、ローカルで動かす場合はサーバー経由で開く必要があります（`file://` 経由では動きません）。
以下のコマンドなどでローカルサーバーを起動してアクセスしてください：

```bash
# Pythonを使用する場合
python -m http.server 8000

# Node.js (npx) を使用する場合
npx serve
```
または、VS Code の拡張機能「Live Server」等をご使用ください。
